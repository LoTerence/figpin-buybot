# Figpin-Buybot

> Python script that imitates the figpin.com web client and allows the user to rapidly purchase items on the figpin online store 

> Tech used: beautifulsoup4


## Quick Start

main.py will run in this order:

1. avCheck
2. addToCart
3. checkOutCart
4. contShipping
5. contPayment
6. buyProduct
